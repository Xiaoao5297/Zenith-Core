## 一、介绍

**Zenith 插件包**是一个面向 **MCPE（Minecraft: Pocket Edition）** 服务器的综合性资源集合，专为 **PMMP** 和 **Nukkit** 核心设计。本资源包整合了数百个精选插件、数十个服务器核心、开服工具及手机 PHP 核心，旨在为服务器主提供一站式解决方案，降低配置门槛，提升开服效率。

### 资源亮点
- **超 600 个** PocketMine 插件！
- **近 130 个** Nukkit 插件！
- **29 个** 精选使用地图
- **50 个** 服务器核心
- 更适合手机的开服软件/核心
- 新手指导 / 常见问题 / 解决方法

### 版本适配
- 大部分插件/核心适配 **MCPE 0.14.x**
- 支持版本范围：**0.11 ~ 1.2**
- 核心类型：PMMP / Nukkit
- 系统环境：Windows / Linux / Android（手机 PHP 核心）

## 二、快速开始

### 1. 环境准备
- 下载本压缩包后解压，您的设备大约需要 **530 MiB** 的空间。
- 确保已安装 Java8（Nukkit）或 PHP（PMMP）运行环境。

### 2. 插件添加方法
1. **关闭服务器**（重要！）
2. 将需要的插件放入服务器 `plugins/` 文件夹下
3. 启动服务器
4. 查看插件是否加载（控制台或日志）
5. **完成！** 尽情使用吧！

### 3. 地图添加方法
1. 确保服务器已安装多世界插件
2. **关闭服务器**（重要！）
3. 解压地图压缩包
4. 重命名文件夹（推荐英文，如 `zc/`、`diyu/`，避免中文）
5. 启动服务器
6. 进入游戏，传送到新世界（通常命令：`/w 世界名称`）
7. 检查世界是否有崩坏/方块异常
8. **完成！** 尽情使用吧！

### 4. 开服方法
详见开服包内的说明文档。

## 三、注意事项

1. **环境依赖**  
   部分插件需特定 PHP 扩展（如 `pthreads`、`yaml`），请确保环境满足要求。

2. **手机 PHP 核心**  
   仅供测试使用，生产环境建议使用标准服务端（如官方 PMMP/Nukkit）。

3. **定期备份**  
   建议定期备份 `worlds/` 和 `playerdata/` 目录，防止数据丢失。

4. **配置修改**  
   修改插件配置前，请阅读对应插件文件夹内的说明文档。

5. **许可证声明**  
   本资源包内部分插件版权归原作者所有，详见各插件文件夹内的许可证文件。
