    =========== Zenith 插件包 ===========
    
    Zenith 插件包, 不只是插件。
    
    一、介绍
    
    1. Zenith 插件包是一个面向 MCPE（Minecraft: Pocket Edition）服务器的综合性资源集合，
    
        专为 PMMP 和 Nukkit 核心设计。本资源包整合了数百个精选插件、数十个服务器核心、开服工具及手机 PHP 核心，
        
        旨在为服务器主提供一站式解决方案，降低配置门槛，提升开服效率。
        
    2. Zenith 插件包内含丰富的内容，不只是插件:
        
        ◎ 超 600 个 PocketMine 插件！
        ◎ 进 130 个 Nukkit 插件！
        ◎ 29 个使用地图
        ◎ 50 个服务器核心
        ◎ 更适合手机的开服软件/核心
        ◎ 新手指导/常见问题/解决方法
        
    3. [资源版本]
        
        ◎ 大部分版本为 0.14 的插件/核心
        ◎ 0.11 ~ 1.2 支持版本的插件
    
    4. 资源来源

    二、快速开始
    
    1. 下载本压缩包后解压，您的设备大约需要 530MiB 的空间来解压文件。
    
    2. 插件添加使用方法:
        
        ◎ 关闭服务器(重要!)
        ◎ 把你需要的插件放入服务器plugins/文件夹下
        ◎ 启动服务器
        ◎ 查看插件是否加载
        ◎ 完成! 尽情使用吧!
    
    3. 地图添加使用方法:
    
        ◎ 确保您的服务器安装了多世界插件
        ◎ 关闭服务器(重要!)
        ◎ 解压地图压缩包
        ◎ 重命名成你想要的文件名(比如zc/, diyu/, 不推荐使用中文)
        ◎ 启动服务器
        ◎ 进入游戏，传送到刚刚添加的世界(不同多世界传送方法不同，通常为/w 世界名称)
        ◎ 检查世界是否有崩坏/方块不对/出现错误方块现象
        ◎ 完成! 尽情使用吧!
        
    4. 开服方法: 详见开服包内。
    
    
    三、注意事项
        
    1. 部分插件需特定 PHP 扩展（如 pthreads、yaml），请确保环境满足要求
    2. 手机 PHP 核心仅供测试，生产环境建议使用标准服务端
    3. 定期备份 `worlds/` 和 `playerdata/` 目录
    4. 修改插件配置前请阅读对应说明文档
