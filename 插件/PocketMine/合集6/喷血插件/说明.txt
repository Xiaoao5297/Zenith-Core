命令

/blood set <实体> <方块ID> C 设置实体喷的什么血(方块)
/blood entities C 列出所有支持的实体
/blood get <实体> C 得到实体喷出的是什么血(方块)
/blood reload C 重新加载配置
/blood help C 列出命令的帮助 (暂时没用v1.0.1+才更新)
