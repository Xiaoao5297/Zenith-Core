>> 请问这是一款什么鬼插件 <<

这是一款能让你在你的服务器里写书的一款插件



>> 写书可以用来干什么 <<

你可以用书来表白 (情书), 用来作死 (遗书), 让熊孩子大哭 (检讨书)

如果您有小游戏服务器, 你也可以用书来写游戏规则啊, 服务器规则啊之类的



>> 怎么写一本书 <<

您需要一个墨囊, 一本书, 和一个羽毛来写哦

当你收集完以上物品后... 您可以输入指令: /book write <书名> <内容>

 

>> 我想在书里换行可以吗 <<

可以啊.. 您可以输入'@&' 进行换行, 当然.. 您可以在配置文件(config.yml)里修改换行符号

 

>> 我可以复制一本书吗 <<

您需要一个墨囊, 一本书, 和一个羽毛, 和手上拿着要复制的书哦

然后... 输入指令: /book copy

 

>> 我是服务器管理, 我可以查看书的内容吗 <<

您可以输入指令: /book query [书ID] 来查看哦!

不填写ID则默认手上的书

  

>> 我是服务器管理员, 有人用书写我坏话怎么办 <<

您可以输入指令: /book delete <书ID> 来删书哦

 

>> 我是服务器管理员, 如何给玩家一本书呢 <<

您可以输入指令: /book give <玩家ID> <书ID> 给玩家一本书哦
