乌兰托娅万岁别点白块儿插件

这是一个游戏插件，主要是游戏

现有功能

快捷地设置游戏平台
方块的移动 C 游戏核心
完成游戏和游戏失败 C 游戏核心
游戏超时判定!(玩家一定时间内不进行操作/退出服务器,自动停止游戏) C 游戏核心
排行榜-目前有Top1,Top2,Top3
玩家不能破坏游戏平台,在游戏平台周围不能放置方块
修复Bug,测试版改为正式版!

未来展望

设置多个游戏平台(让更多的玩家同时进行游戏)
数据库 C 记录玩家成绩(乌兰托娅已做好，但正在完善）
设置更多排行榜
正在进行游戏的玩家不会被打扰
结合经济插件-游戏奖励（有，但现在无法结合经济插件）
做出一个4*5的方块，3个牌子

使用方法

安装命令/dttt set
点击4*5方块左下角第一点
点击4*5方块右上角地4点
点击3个牌子
卸载命令/dttt del
